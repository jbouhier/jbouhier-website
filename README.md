# jbouhier.com

Personal website

## Commands

| Command                   | Action                                    |
| :------------------------ | :---------------------------------------- |
| `npm install`             | Install dependencies                      |
| `npm run dev`             | Starts local dev server `localhost:4321`  |
| `npm run build`           | Build production app to `./dist/`         |
| `npm run preview`         | Preview production app locally            |
| `npm run astro ...`       | Run other Astro commands                  |
| `npm run astro -- --help` | Astro CLI help                            |
